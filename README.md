# 國教院工作爬蟲
利用google sheet apps script來定時爬國教院新職缺，並記錄起來。

## 功能
1.定時爬取國教院工作資訊。

2.存入google sheet當歷史資料。

3.將十五天內的工作資訊傳到line通知。

## 特色
1.只要有google帳號即可使用。

2.免架設伺服器。

3.在google應用程式中完成自動通知功能。
